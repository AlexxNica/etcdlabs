Manage etcd with Kubernetes


Application container gives much portability to initial development phase. But operating and scaling container requires much more work. Kubernetes manages such complex container environments. This is an example workflow to manage etcd with Kubernetes, in most manual way possible; see Running Kubernetes Locally via Minikube and other guides below for easier ways.
